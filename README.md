# دورة Six Sigma Green Belt — GitHub Pages

## الرفع من الآيباد أو الكمبيوتر
1. أنشئ مستودع GitHub جديداً واجعله Public.
2. ارفع **جميع الملفات الموجودة في هذا المجلد** إلى جذر المستودع، وليس ملف index.html وحده.
3. افتح: Settings → Pages.
4. تحت Build and deployment اختر: Deploy from a branch.
5. اختر branch: main، والمجلد: / (root)، ثم Save.
6. انتظر دقيقة أو دقيقتين. سيظهر الرابط بهذا الشكل:
   `https://USERNAME.github.io/REPOSITORY/`

## التثبيت على الآيباد
افتح رابط الموقع في Safari، ثم:
Share → Add to Home Screen

بعد أول فتح ناجح، يستطيع الموقع العمل دون إنترنت في أغلب الحالات بفضل Service Worker.

## الملفات
- `index.html`: الموقع والدروس والاختبارات.
- `manifest.webmanifest`: إعدادات التثبيت كتطبيق.
- `service-worker.js`: التخزين المؤقت والعمل دون إنترنت.
- `icon-192.png` و`icon-512.png`: أيقونات التطبيق.
