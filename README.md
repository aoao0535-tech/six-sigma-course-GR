# Six Sigma Green Belt — Language Button Fix (v5)

تم إصلاح زر اللغة ليتوافق مع Safari والآيباد:

- زر `🌐 English` يحول الموقع كاملاً إلى الإنجليزية.
- زر `🌐 العربية` يعيده إلى العربية.
- أزيل أي عنصر زخرفي كان يمكن أن يغطي الزر ويمنع اللمس.
- تم تحديث التخزين المؤقت حتى تظهر النسخة الجديدة.

## الاستبدال في GitHub
1. ارفع جميع الملفات الموجودة في هذه الحزمة فوق الملفات القديمة.
2. اضغط **Commit changes**.
3. افتح الموقع بإضافة `?v=5` إلى نهاية الرابط أول مرة.

مثال:
`https://USERNAME.github.io/REPOSITORY/?v=5`
