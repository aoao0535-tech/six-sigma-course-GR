# Six Sigma Green Belt — Arabic / English

الموقع يحتوي على زر واضح أعلى الصفحة:

- عند ظهور الموقع بالعربية اضغط `🌐 English` للتحويل إلى الإنجليزية.
- عند ظهور الموقع بالإنجليزية اضغط `🌐 العربية` للعودة إلى العربية.
- اختيار اللغة يُحفظ تلقائياً على جهاز المستخدم.

## تحديث الموقع على GitHub
ارفع جميع الملفات الموجودة في هذه الحزمة إلى جذر المستودع، ثم اختر **Commit changes**.
لا تحتاج إلى إنشاء مستودع جديد أو تغيير رابط GitHub Pages.

إذا ظهرت النسخة السابقة في Safari، افتح الرابط مرة باستخدام:
`?v=4`
في نهاية رابط الموقع.
